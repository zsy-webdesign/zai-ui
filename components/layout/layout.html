Layout 布局 ​

  • 协助进行页面级整体布局。

组件概述 ​

  • ZaiLayout：布局容器，其下可嵌套 ZaiHeader ZaiSider ZaiContent ZaiFooter 或 ZaiLayout 本身，可以放在任何父容器中。
  • ZaiHeader：顶部布局，自带默认样式，其下可嵌套任何元素，只能放在 ZaiLayout 中。
  • ZaiSider：侧边栏，自带默认样式及基本功能，其下可嵌套任何元素，只能放在 ZaiLayout 中。
  • ZaiContent：内容部分，自带默认样式，其下可嵌套任何元素，只能放在 ZaiLayout 中。
  • ZaiFooter：底部布局，自带默认样式，其下可嵌套任何元素，只能放在 ZaiLayout 中。
