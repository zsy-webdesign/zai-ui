Decorate 装饰 ​

基础用法 ​

使用'type''来定义按钮样式。

type为1 ​

<template>
  <div class="decorate-container">
    <ZaiDecorate style="height: 50px; width: 300px" type="1"></ZaiDecorate>
    <ZaiDecorate
      style="height: 50px; width: 500px; margin-top: 24px"
      type="1"
      lightColor="red"
      darkColor="yellow"
    ></ZaiDecorate>
  </div>
</template>
<style scoped>
.decorate-container {
  display: flex;
  flex-direction: column;
  align-items: center;
}
</style>

darkColor深色主题颜色false
