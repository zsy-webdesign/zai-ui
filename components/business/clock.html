Clock 不同国家时钟 ​

  • 暂时只支持深色背景。

基础用法 ​

  • 下面例子中在外层包裹了黑色背景的div，是为了演示深色背景下的效果。
<template>
  <div class="zai-clock-demo">
    <div class="demo-row">
      <ZaiClock class="demo-item" countryCode="cn" timeZone="1"></ZaiClock>
      <ZaiClock class="demo-item" countryCode="fr" timeZone="8"></ZaiClock>
      <ZaiClock class="demo-item" countryCode="af" timeZone="4"></ZaiClock>
      <ZaiClock class="demo-item" countryCode="ar" timeZone="-3"></ZaiClock>
    </div>
    <div class="demo-row">
      <ZaiClock
        class="demo-item"
        width="196"
        countryCode="mt"
        timeZone="-7"
      ></ZaiClock>
      <ZaiClock
        class="demo-item"
        width="196"
        countryCode="et"
        timeZone="-5"
      ></ZaiClock>
      <ZaiClock
        class="demo-item"
        width="196"
        countryCode="pw"
        timeZone="9"
      ></ZaiClock>
    </div>
    <div class="demo-row">
      <ZaiClock
        class="demo-item"
        width="256"
        countryCode="fj"
        timeZone="12"
      ></ZaiClock>
      <ZaiClock
        class="demo-item"
        width="256"
        countryCode="hk"
        timeZone="8"
      ></ZaiClock>
      <ZaiClock
        class="demo-item"
        width="256"
        countryCode="mm"
        timeZone="6.5"
      ></ZaiClock>
    </div>
  </div>
</template>
<style lang="less" scoped>
.zai-clock-demo {
  .demo-row {
    display: flex;
    justify-content: space-around;
    margin-bottom: 50px;
    .demo-item {
      background: #1b1b1f;
    }
  }
}
</style>

属性值 ​

属性名说明类型默认值必传
countryCode国家代码 点击可查看,可参考过去国旗网站v-model string'cn'true
timeZone时区 东8区传8，西2区传-2 点击可查看string'8'true
width宽度 正方形，长宽相同string'108'true

备注：各个国家国旗
